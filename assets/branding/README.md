# Branding

Assets relacionados à marca Conte os Feitos.

- `logos/`: originais, versões aprovadas e histórico de logos;
- `app-icon/`: estudos e exportações do ícone do aplicativo;
- `favicon/`: variações de favicon;
- `splash/`: telas e artes de abertura;
- `social/`: peças e modelos para redes sociais;
- `mockups/`: simulações de aplicação da marca.

Somente arquivos em estado aprovado devem ser usados em materiais finais.
