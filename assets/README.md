# Assets

Repositório de trabalho para identidade visual, ilustrações, elementos de interface e registros visuais do Conte os Feitos.

Esta pasta não é servida automaticamente pela aplicação. Assets só devem ser copiados para `public/` em uma tarefa funcional específica, revisada e testada.

- `branding/`: marca e materiais institucionais;
- `illustrations/`: ilustrações de produto e módulos;
- `ui/`: elementos gráficos reutilizáveis da interface;
- `screenshots/`: registros visuais seguros para documentação e divulgação.

Consulte [docs/DESIGN/ASSET_GUIDE.md](../docs/DESIGN/ASSET_GUIDE.md) antes de adicionar novos arquivos.
