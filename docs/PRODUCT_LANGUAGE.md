# Linguagem do produto

| Conceito | Nome oficial |
| --- | --- |
| Marca | Conte os Feitos |
| Produto | Quiz Bíblico |
| Competição semanal | Jornada |
| Modo de prática | Jornada de Treino |
| Questões | Perguntas |
| Repositório reutilizável de conteúdo | Acervo de perguntas |
| Participação | Tentativas |
| Classificação | Ranking |
| Conquistas | Medalhas |
| Período | Temporada |

Na interface destinada ao participante, **Jornada** substitui “Rodada” ou “semana” quando representam a competição semanal. **Acervo de perguntas** é o nome preferencial para o repositório reutilizável de conteúdo. Nomes internos de tabelas, APIs, tipos e classes podem continuar usando `round`, `rodada` ou `question_bank` para preservar compatibilidade técnica.

Documentos técnicos podem usar “rodada” ou “banco” ao citar tabelas, rotas, migrations ou nomes internos. Documentos operacionais e a interface voltada ao participante devem preferir “Jornada” e “Acervo de perguntas”.
