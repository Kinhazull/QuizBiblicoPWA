# Decisões

Reúne decisões duradouras de produto, design e arquitetura. Decisões gerais já registradas permanecem em `docs/PRODUCT/DECISION_LOG.md`; registros arquiteturais formais e independentes devem ser adicionados em `ADR/`.

Uma decisão não deve ser apagada quando substituída. O novo registro deve apontar qual decisão anterior foi superada e explicar o motivo.
